# 仕様
動作確認は最終版であるRalpha_170111のみで行っております。
Ralpha.exeのあるディレクトリの上位の階層のディレクトリを"Ralpha"にしてください。
Ralpha.exeのあるディレクトリの上位の階層のディレクトリの"Ralpha"と同じ階層にSetup.exeを置いてください。
Ralpha.exeは管理者権限で実行してください。ユーザー権限ですと不具合が発生します。
